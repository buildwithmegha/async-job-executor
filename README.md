# Async Job Executor (Spring Boot + Multithreading)

A backend project demonstrating multithreading concepts using Spring Boot.

## Tech Stack
- Java 17
- Spring Boot 3
- H2 Database
- Spring Data JPA

## Current Features (Day 1)
- Create Job API (POST /jobs)
- Fetch Job by ID (GET /jobs/{id})
- List Jobs (GET /jobs)

## Upcoming (Day 2+)
- Thread pool executor
- Job queue and dispatcher
- Async job execution
- Cancellation, retries, monitoring
